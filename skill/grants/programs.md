# Solana Grant Programs (2025–2026)

## 1. Solana Foundation Developer Grants

**Funder:** Solana Foundation  
**Focus:** Core infrastructure, developer tooling, education, open-source public goods  
**Amounts:** $5,000 – $100,000+ (milestone-based disbursement)  
**Timeline:** Rolling — decisions in 4–8 weeks  
**URL:** https://solana.org/grants  

**Best for:**
- Open-source libraries and SDKs
- Developer education and documentation
- Protocol-level tooling (validators, RPC, indexers)
- Research with public benefit

**Not for:** Closed-source commercial products, token launches, pure marketing

---

## 2. Superteam Grants

**Funder:** Superteam (regional chapters + central)  
**Focus:** Products and projects that grow the regional Solana ecosystem  
**Amounts:** $500 – $10,000 (most under $5,000)  
**Timeline:** Rolling — decisions in 1–3 weeks  
**URL:** https://superteam.fun/grants  

**Best for:**
- Early-stage projects with regional traction
- Community tools, hackathon follow-ups
- Builders already active in a Superteam region

**Not for:** Abstract research, projects without a clear Solana use case

---

## 3. Colosseum Accelerator / Grants

**Funder:** Colosseum  
**Focus:** Post-hackathon teams graduating to funded startups  
**Amounts:** $100,000 – $250,000 (equity investment via accelerator)  
**Timeline:** Cohort-based — apply after winning or placing in a Colosseum hackathon  
**URL:** https://www.colosseum.org  

**Best for:**
- Hackathon winners looking to go full-time
- Teams with a working prototype and clear PMF hypothesis

**Not for:** Pre-idea stage, teams without a hackathon track record

---

## 4. Helius Builder Grants

**Funder:** Helius  
**Focus:** Projects that use Helius APIs in production (RPC, webhooks, DAS API)  
**Amounts:** $1,000 – $25,000 in API credits + cash  
**Timeline:** Rolling  
**URL:** https://helius.dev/grants  

**Best for:**
- Indexers, explorers, analytics dashboards
- DeFi dashboards consuming on-chain data
- NFT tooling that relies on DAS API

**Not for:** Projects that don't use Helius infrastructure

---

## 5. Metaplex Foundation Grants

**Funder:** Metaplex Foundation  
**Focus:** NFT infrastructure, digital ownership, creator tools  
**Amounts:** $5,000 – $50,000  
**Timeline:** Rolling, reviewed quarterly  
**URL:** https://www.metaplex.com/grants  

**Best for:**
- NFT tooling, marketplaces, creator platforms
- Token Metadata / Core / Bubblegum integrations
- Digital ownership primitives

**Not for:** Generic DeFi with no NFT component

---

## 6. Phantom / Wallet Ecosystem Grants

**Funder:** Phantom  
**Focus:** Wallet UX improvements, browser extension integrations, mobile tooling  
**Amounts:** $2,000 – $20,000  
**Timeline:** Rolling  
**URL:** Contact ecosystem@phantom.app  

**Best for:**
- Dapp developers improving wallet connect flows
- Mobile wallet adapter improvements
- Security tooling for wallet users

---

## 7. Wormhole Foundation Grants

**Funder:** Wormhole Foundation  
**Focus:** Cross-chain applications, bridge tooling, interoperability  
**Amounts:** $10,000 – $250,000  
**Timeline:** Rolling  
**URL:** https://wormhole.com/grants  

**Best for:**

- Multi-chain NFT platforms

**Not for:** Solana-only projects with no cross-chain component

## 8. Jupiter Grant Program (JUP DAO)

**Funder:** JUP DAO / Jupiter  
**Focus:** DEX aggregation improvements, liquidity infrastructure, DeFi tooling  
**Amounts:** Variable — JUP token grants via governance  
**Timeline:** Governance vote cycle (~2–4 weeks per cycle)  
**URL:** https://vote.jup.ag  

**Best for:**
- Liquidity routing improvements
- DeFi tooling that integrates Jupiter
- Open-source DEX research
